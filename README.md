# 🌦️ Skycaster – Weather App

A modern and responsive **weather application** that provides real-time weather data for any city in the world using the WeatherAPI.

---

## 🚀 Live Demo

🔗 https://raghavpandey92.github.io/Weather-app/

---

## 📌 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature (°C & °F)
* ☁️ Weather condition with icons
* 💨 Wind speed & direction
* 💧 Humidity
* 👀 Visibility
* ☀️ UV Index with levels
* 🌫️ Air Quality Index (AQI) with status
* ⚡ Fast and responsive UI
* 🎨 Modern glassmorphism design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Weather API

---

## 📂 Project Structure

```
Weather-app/
│── index.html
```

---

## ⚙️ How It Works

1. User enters a city name
2. App sends request to WeatherAPI
3. API returns real-time weather data
4. Data is displayed dynamically on UI

---

## 🔑 API Used

* https://api.weatherapi.com/
